1. 产品介绍

LingVa 是一款轻量、免费、开源的 Chrome 浏览器翻译插件，支持以下核心功能：

  • 弹窗翻译：点击工具栏图标，在弹窗中输入文字进行翻译。
  • 划词翻译：在任意网页选中文字，点击浮出的翻译按钮即可。
  • 多引擎支持：内置免配置翻译引擎，开箱即用；也可接入百度翻译、Google 翻译、微软翻译、有道翻译，或配置自己的翻译后台接口。
  • 翻译历史：自动保存最近 100 条翻译记录，弹窗、悬浮球、划词翻译三个入口共用同一份历史与常用语言列表。
✓ 内置翻译引擎开箱即用，无需任何配置，安装后直接使用。如需更高用量或更稳定的服务，可在「设置 → 翻译引擎」中切换至第三方引擎并填入 API 凭证。

2. 弹窗翻译

点击浏览器工具栏右上角的 LingVa 图标，即可打开翻译弹窗。

  1. 1
    选择语言
    左侧下拉框选择源语言（支持「自动检测」），右侧选择目标语言；翻译过的目标语言会出现在输入框上方的「常用语言」快捷按钮中，点击即可快速切换。
  2. 2
    输入文字
    在文本框中输入或粘贴需要翻译的内容，最长 1024 个字符。
  3. 3
    点击翻译
    点击「翻译」按钮，稍等片刻即可在下方看到翻译结果。
  4. 4
    复制或重新翻译
    点击结果区域下方的「复制」按钮，将翻译内容复制到剪切板；旁边的刷新图标可用当前文本重新发起一次翻译。
ℹ 翻译历史记录会在弹窗底部展示（悬浮球、划词翻译产生的记录也会一并汇总在这里），点击每条记录可重新加载该次翻译，点击右侧的关闭图标可删除单条记录；暂无记录时会显示空状态提示。

3. 划词翻译

在任意网页上，用鼠标选中一段文字（或用键盘 Shift + 方向键选中），即可触发划词翻译功能。

  1. 1
    选中文字
    在网页上拖拽鼠标选中需要翻译的文字。
  2. 2
    点击翻译按钮
    松开鼠标后，选中文字附近会浮现「翻译」按钮，点击即可。
  3. 3
    查看结果
    翻译结果显示在浮动面板中，按住结果文字区域可拖拽移动面板位置；面板下方可切换目标语言、复制结果，点击图钉图标可将面板钉住，钉住后点击面板外部不会自动关闭。

关闭划词翻译：如果不需要该功能，可在「设置 → 常规设置」中开启「禁用划词翻译」，开启后在任意网页选中文字将不再弹出翻译按钮或翻译面板。

翻译成功后会自动计入弹窗底部的翻译历史，所选目标语言也会同步更新到弹窗的「常用语言」快捷按钮中。

4. 悬浮球

悬浮球是一个常驻在网页角落的快捷入口，点击后可打开独立翻译面板，无需切换回插件弹窗即可翻译。在「设置 → 常规设置」中开启「显示悬浮翻译球」，悬浮球将出现在所有匹配的网页上。

  1. 1
    打开面板
    点击悬浮球打开翻译面板；再次点击悬浮球会将面板最小化，已输入的内容不会丢失。
  2. 2
    输入文字
    在面板的输入框中输入或粘贴文字，停顿片刻后自动发起翻译，无需额外点击。
  3. 3
    查看结果
    翻译结果显示在下方结果区域，可切换目标语言、点击刷新图标重新翻译、点击「复制」按钮复制结果。
  4. 4
    移动或钉住面板
    按住标题栏可拖拽移动面板位置；点击标题栏右侧的图钉图标可将面板钉住，钉住后点击面板外部不会自动关闭。

翻译成功后同样会计入弹窗底部的翻译历史与「常用语言」快捷按钮，与弹窗、划词翻译共用同一份记录。

⚠ 部分网站的内容安全策略（CSP）可能阻止悬浮球显示，属正常现象。

5. 翻译引擎配置

LingVa 提供一个开箱即用的内置翻译引擎、四个可选的第三方引擎，以及自定义接口功能，支持接入您自己的翻译后台服务。所有引擎共用同一套配置界面：在「设置 → 翻译引擎」页面直接点击引擎卡片即可切换为当前使用的引擎，同一时间只有一个引擎生效；未配置的引擎点击卡片上的「设置」按钮展开配置表单，自定义接口则在「设置 → 自定义接口」页面单独配置。

内置翻译 无需配置 默认

基于 Google 翻译服务，安装后开箱即用，无需注册账号或填写任何 API 密钥。适合绝大多数日常翻译场景。

⚠ 内置翻译使用非官方接口，Google 未对其提供稳定性保证，在极少数情况下可能出现请求失败。如需更稳定或更高用量的服务，建议切换至下方任意第三方引擎并配置 API 凭证。

自定义接口 高级功能

如果您有自建翻译服务（本地 LLM、私有 API 网关、企业翻译平台等），可以在「设置 → 自定义接口」中填入接口地址和可选的 Token，LingVa 将按照以下协议规范向该地址发起翻译请求。

接口规范

项目说明
请求方法POST
Content-Typeapplication/json
认证方式可选；填写 Token 后自动附加 Authorization: Bearer {Token} 请求头
请求体字段 q待翻译的原始文本
请求体字段 from源语言 BCP-47 代码，自动检测时值为 "auto"
请求体字段 to目标语言 BCP-47 代码，如 zh-Hans、en、ja
成功响应{ "dst": "翻译结果" }
错误响应{ "error": "错误说明" }
ℹ 保存配置后，LingVa 会自动切换为自定义接口引擎。如需切回其他引擎，在「设置 → 翻译引擎」中点击「切换」即可。

百度翻译 免费额度充足

百度 AI 开放平台提供每月 100 万字符的免费翻译额度，适合日常使用。

  1. 1
    注册账号
    前往 百度 AI 开放平台 注册并登录。
  2. 2
    开通机器翻译服务
    进入「产品服务」→「机器翻译」，点击「立即使用」开通服务。
  3. 3
    创建应用
    在控制台创建一个应用，勾选「机器翻译」能力，提交后获得 App ID 和 API Key。
  4. 4
    填入插件
    在 LingVa「设置 → 翻译引擎 → 百度翻译 → 设置」中填入：
    App ID 对应控制台的 App ID
    Bearer Token 对应控制台的 API Key

Google 翻译 翻译质量高

Google Cloud Translation API 翻译质量优秀，每月提供 50 万字符免费额度，超出后按量计费。

  1. 1
    创建 Google Cloud 项目
    前往 Google Cloud Console，创建一个新项目（或使用已有项目）。
  2. 2
    启用 Translation API
    在「API 和服务」→「库」中搜索「Cloud Translation API」并启用。
  3. 3
    创建 API 密钥
    进入「API 和服务」→「凭据」，点击「创建凭据」→「API 密钥」，复制生成的密钥。
  4. 4
    填入插件
    在 LingVa「设置 → 翻译引擎 → Google 翻译 → 设置」中填入 API 密钥。
⚠ 使用 Google Cloud 需绑定付款方式。建议在 API 密钥上设置配额限制，防止意外超额计费。

微软翻译 免费额度充足

Azure AI 翻译服务每月提供 200 万字符的免费翻译额度（F0 免费层），翻译质量优秀，支持 100+ 语言。

  1. 1
    注册 Azure 账号
    前往 Azure 门户 注册并登录（需绑定信用卡，F0 层不收费）。
  2. 2
    创建 Translator 资源
    在「创建资源」中搜索「Translator」，选择免费定价层 F0（2M 字符/月），选择区域（如 East Asia）后创建。
  3. 3
    获取密钥和区域
    资源创建完成后，进入「密钥和终结点」页面，复制「密钥 1」和「位置/区域」字段。
  4. 4
    填入插件
    在 LingVa「设置 → 翻译引擎 → 微软翻译 → 设置」中填入：
    Subscription Key 对应「密钥 1」
    Region 对应「位置/区域」，如 eastasia
⚠ 区域字段须填写 Azure 区域标识符（英文小写），例如 eastasia、eastus、westeurope，填写错误将导致请求失败。

有道翻译 支持多语言

有道智云文本翻译 API 支持 100+ 语种互译，新注册用户享有免费体验额度。

  1. 1
    注册账号
    前往 有道智云开放平台 注册并登录。
  2. 2
    开通文本翻译服务
    进入「自然语言翻译」→「文本翻译」，点击「立即使用」开通服务并充值体验包（或购买）。
  3. 3
    创建应用
    在控制台「应用管理」中创建新应用，绑定「文本翻译」服务，提交后获得 App Key 和 App Secret。
  4. 4
    填入插件
    在 LingVa「设置 → 翻译引擎 → 有道翻译 → 设置」中填入：
    App Key 和 App Secret（均来自应用管理页面）

6. 权限与隐私说明

LingVa 本身不收集、不存储、不上传任何个人数据。安装时 Chrome 会展示所需权限，以下逐项说明用途：

权限用途
storage将 API 密钥、翻译历史、偏好设置保存在本机，不涉及任何远程存储。
activeTab获取当前标签页的选中文字，用于划词翻译，不读取页面其他内容。
scripting向网页注入划词翻译与悬浮球所需的脚本，仅在用户触发时运行。
offscreen在后台离屏文档中发起翻译 API 请求，绕过 MV3 对 Service Worker 的网络限制。
host_permissions划词翻译需要在所有网页上注入内容脚本，以及向各翻译引擎 API 发起跨域请求。

翻译文本的去向：您输入的文字会直接从浏览器发送至当前使用的翻译引擎（内置翻译 / 百度 / Google / 微软 / 有道 / 您配置的自定义接口），LingVa 不经手任何翻译请求，也不设中间服务器。

ℹ 您可以通过 Chrome 扩展开发者工具（F12 → Network / Sources）自行审查插件的全部网络请求与源码，无需信任任何承诺。

完整的数据处理说明请参阅 隐私权政策。

显示语言切换插件界面语言，目前支持中文和英文。
字体大小拖动滑块在 12–20px 范围内连续调整翻译结果的字体大小。
禁用划词翻译开启后，在任意网页选中文字将不再弹出翻译按钮或翻译面板。

恢复默认设置一键将显示语言、字体大小、禁用划词翻译、显示悬浮翻译球、主题模式恢复为默认值。
